Exploring AWS S3 & CLI — From Bucket Creation to Public Access
I recently explored working with Amazon S3 using the AWS CLI. The goal was to:
Create a new S3 bucket
Upload a sample text file using CLI
Make the file publicly accessible via a browser
Along the way, I navigated key AWS features like block public access settings, object ACLs, and bucket policies — understanding how they interact and restrict/allow public access. With the right configuration, I was finally able to browse the uploaded file directly via its public URL.
This hands-on session helped me better grasp S3’s permission model and the importance of IAM roles and policies in managing cloud resources securely.
